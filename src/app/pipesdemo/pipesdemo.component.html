Pipes

Pipes are referred as filters. It helps to transform data and manage data within interpolation, denoted by | symbol .

It accepts data, arrays, integers and strings as inputs which are separated by | symbol.

Uppercase Pipe

{{title | uppercase}}

Lowercase Pipe

{{title | lowercase}}

Currency Pipe

{{6589.23 | currency: "INR"}}
{{6589.23 | currency: "USD":true}}

Date Pipe

{{todaydate | date:'d/M/y'}}
{{todaydate | date:'shortTime'}}
{{todaydate | date:'fullDate'}}

Decimal Pipe

{{454.78787814 | number : '3.4-4' }}

Json Pipe

{{ jsonval | json }}

Percent Pipe

{{00.54565 | percent}}

Slice Pipe

{{months | slice:2:6}} // here 2 and 6 refers to the start and the end index
